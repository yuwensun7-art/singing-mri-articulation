## Data description

This directory documents the source MRI material and derived data representations used in the study.

The real-time MRI recordings analyzed originate from publicly available video material hosted on YouTube:

https://www.youtube.com/watch?v=REHXN-idz-I

As the source recordings are not owned by the author, they are not redistributed here.

All shared materials consist of derived representations and numerical results used for analysis and reproducibility.
